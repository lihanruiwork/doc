# 语法分析之消除左递归
* 公式
```
A -> Aα | Aβ | γ
```
转化为
```
A -> γR
R -> αR | βR | ε
```
* 例子(加减法)
```
expr -> expr + term
     |  expr - term
     |  term
term -> 0
     |  1
       ...
     |  9  
```
公式对照
```
γ = term
A = expr
& = + term
β = - term

so 

expr -> term R
R -> +term R 
  |  -term R
  |  ε
term -> 0
     |  1
       ...
     |  9
```
替换一下R
```
expr -> term rest
rest -> +term rest 
     |  -term rest 
     |  ε
term -> 0
     |  1
       ...
     |  9
```
